# Elden Ring Basic Build Calculator

This example demonstrates a simplified Elden Ring character build calculator. It allows users to adjust character attributes (Vigor, Strength, Dexterity, Endurance) and select basic equipment (weapon, armor pieces). The calculator then dynamically computes key character statistics such as total HP, weapon attack rating (AR), and current equip load with its corresponding status.

## Language

`html`

## How to Run

Save the code as `index.html`. Open `index.html` in any modern web browser.
